## 功能：

 1）多线程 

 2）自动过验证码 

 3）每个人可以绑定多个uid

 4）每个uid可以设置不同的推送内容

 5）内存记录击剑记录，重启bot会清空

####  6）大部分指令仅支持群聊，支持私聊/群聊推送。
